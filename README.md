# Pedro Más · Land Clearing Services

Bilingual landing page (Spanish/English) for professional land clearing and brush cutting services in Padul and Lecrin Valley (Granada).

## 🚀 Deployment (Vercel)

If you are deploying this project to **Vercel**, please ensure you configure the following:

1. **Root Directory**: Select the `frontend` folder.
2. **Framework Preset**: Other (or None).
3. **Output Directory**: `.` (Current directory inside `frontend`).

## 📁 Project Structure

- `frontend/`: All public web content.
  - `index.html`: Main Spanish version.
  - `en/index.html`: English version.
  - `assets/`: Optimized images.
  - `css/`: Stylesheets.
  - `js/`: Interactivity (Theme toggle, etc.).
- `backend/`: Future API implementations or form handlers.

## 💻 Local Development

To run this project locally with a real web server:

1. Install dependencies (if any): `npm install`
2. Run the dev server: `npm run dev`
3. Open: `http://localhost:8080`
